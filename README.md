Pate
----

### Instalation

        git clone https://github.com/pate-wsi/pate.git
        cd pate
        pip install flask flask-sqlalchemy flask-login
        cp server-template.ini server.ini
        vim server.ini
        python setup_app.py
        python runserver.py


### Requirements

  - flask
  - flask-sqlalchemy
  - flask-login
